__NOTE !! Do not use Docker Compose..__
U may use Uvicon to deploy your Python Code. 

Let me know if u need any helps. 

## requirement.txt is all packages u need to install.
> pip install -r requirements.txt

## Get Start with Uvicon

to Start appCategoryServices
> uvicorn app.appCategoryServices:app --reload

to Start appMiddlewareServices
> uvicorn app.appMiddlewareServices:app --reload

to Start appUserServices
> uvicorn app.appUserServices:app --reload
